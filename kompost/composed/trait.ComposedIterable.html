Source

fn composed( self, init_fs: Init, ) -> ComposedIterator<IterIn, IterOut, In, Out, Init> ⓘ

Allows defining grouping frequently used Iterator methods (such as map or scan) and reuse them.

§Arguments
  • init_fn: The closure that receives the current IntoIterator and produces the initial context.
§Examples
use kompost::ComposedIterable;

fn favorite_pipeline(it: impl Iterator<Item = i32>) -> impl Iterator<Item = f64> {
    it.skip(5)
        .map(|x| x.pow(2))
        .take_while(|x| *x < 100)
        .map(|x| x as f64)
}

assert_eq!(
    [1, 2, 3, 4, 5, 6, 7].into_iter().composed(favorite_pipeline).collect::<Vec<_>>(),
    vec![36.0f64, 49.0]
)
